什么是 Iress 交易平台？

Iress Essential 是由 Iress 交易平台 (Iress Trader) 开发的完整的 Iress ViewPoint 软件包，是一个基于 Web 的在线交易平台，可跨多种产品类型和交易所动态传输市场数据。 通过将访问完整的市场深度、增强的内部图表系统与实时新闻流相结合，Iress Trader 为您提供保持市场优势所需的一切。

如何使用 Iress 交易平台？

了解如何使用 Iress 交易平台：

  • • 打开您的 Iress 交易平台（Iress Trader）
  • • 点击屏幕左上角的“分析 (Analytics)”
  • • 选择“投资组合 (Portfolio)”，然后在“投资组合 (Portfolio)”窗口点击“N”按钮
  • • 在“账户导航 (Account Navigator)”弹出窗口中输入您的交易帐号
  • • 通过点击突出显示您的帐号，然后点击“OK (确定)”
  • • 点击“CFD”
  • • 进入深度面板 (Depth Panel) 查看市场深度
  • • 选择您想交易或查看的资产
  • • 右键单击该资产
  • • 创建买单
  • • 自定义您的交易，然后点击“买入”

什么是 Iress ViewPoint？

Iress ViewPoint 是一个新的改进的基于网络的交易解决方案平台。 Iress ViewPoint 是一个综合性的活跃交易者平台，具有先进的功能，可改善交易者的体验。 新的 Iress ViewPoint 提供增强的交易工具和与各种互联网浏览器的更好兼容性。 凭借将数据相互叠加的高级功能，Iress ViewPoint 提供数据可见性，以便交易者可以在交易机会出现时充分利用它们。
