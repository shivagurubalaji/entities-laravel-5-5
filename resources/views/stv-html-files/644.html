Check Point Software Technologies Ltd (CHKP) Profile

Check Point Software Technologies Ltd. is a software company that develops, promotes, and supports a variety of products and services for information technology security around the world. The organisation provides a variety of network security, endpoint security, data security, and managed services to meet the needs of customers. In addition, it offers Check Point Infinity Architecture, a cyber security architecture that protects against 5th and 6th generation cyber-attacks across various networks, endpoints and cloud workloads, the Internet of Things, and mobile; Check Point Security Protocols, security gateways and software platforms that endorse small company and large enterprise data centres and telco-grade environments; and the Check Point SandBlast family, which provides threat prevention and zero-day safeguards. Check Point also provides cloud security products such as Check Point CloudGuard, which provides powerful security, cloud visibility, cloud security posture strategic planning, and workload protection solutions for enterprise cloud networks, data, and applications; Check Point Harmony, which provides security for remote access and availability; and Check Point Security Management, which enables customers from single offices to hundreds of thousands of offices to manage and tailor their security policies. As well as providing technical customer care strategies and activities, Check Point also provides consulting and specialist services in the areas of effective implementation, upgrading and optimising Check Point products, including design planning, security implementation, as well as certification and educational training services for Check Point products. It sells its products and services to companies, service providers, small and medium enterprises, and customers through a network of channel partners, which includes distributors, resellers, system integrators, producers, and managed security service providers, among other entities. Check Point Software Technologies Ltd. is an Israeli company with its headquarters in Tel Aviv.
