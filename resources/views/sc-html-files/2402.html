포렉스에서 VPS 트레이딩은 무엇인가요?

가상개인서버 트레이딩은 흔히 VPS 트레이딩으로 알려져있으며 높은 네트워크 안정성, 정전 및 짧은 지연 시간 실행을 보장하기 위해 독립적인 기능 소프트웨어와 자율 가상 서버 호스팅을 사용하여 포렉스 거래하고 있습니다. 트레이더는 VPS 트레이딩을 자동매매(EA)를 포함한 자동 알고리즘 전략, 장기적으로 고객에게 상당한 비용이 발생할 수 있는 슬리피지를 피하기위한 오프라인 모드에 이용할 수 있습니다.

VPS와 MetaTrader 4, MetaTrader 5 혹은 cTrader 를 어떻게 연결하나요?

Windows용 Forex VPS를 MetaTrader 4, MetaTrader 5 혹은 cTrader 연결하는 방법:

  • Windows 시작 단추로 이동하여 "Windows 액세서리"를 선택합니다.

  • RDP(원격 데스크톱 연결) 버튼을 클릭

  • "FP Markets Forex VPS" IP 주소 및 포트 번호 입력

  • "연결" 버튼을 선택

  • 정보 입력 후 '"확인" 클릭

  • SSL 인증서 경고 메시지에서 "이 컴퓨터에 대한 연결을 다시 묻지 않음" 확인란을 클릭

  • "예" 버튼을 클릭하고 연결

MAC OS용 MetaTrader 4(MT4) 또는 MetaTrader 5(MT5)에 Forex VPS를 연결하는 방법

  • 앱 스토어에서 "Microsft 원격 데스크톱" 응용 프로그램 설치

  • "Microsft 원격 데스크톱" 응용 프로그램을 열고 "새로 만들기"를 클릭합니다. "연결 이름" VPS의 호스트 이름을 입력하십시오 "PC 이름" FP 시장 Forex VPS IP 주소 및 포트 번호 입력 게이트웨이 "구성된 게이트웨이가 없습니다". 자격 증명을 입력하고 탭을 닫습니다

  • “FP Markets Forex VPS” 더블 클릭

  • 팝업에서 "인증서 표시" 단추 클릭

  • "IP 어드레스 연결 시 항상 내 FP Markets Forex VPS 신뢰" 상자를 선택하고 "계속"을 클릭

  • 컴퓨터의 인증 정보 입력

  • "업데이트 설정" 단추 클릭 후 연결

Android 또는 iOS 에서 MetaTrader 4(MT4) 또는 MetaTrader 5(MT5)에 Forex VPS를 연결하는 방법

모바일 앱에서 Forex 원격 데스크톱 프로토콜 "RDP"를 사용하고 FP Markets Forex VPS에서 제공하는 서버 세부 정보를 사용
