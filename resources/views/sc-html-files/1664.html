Pares de divisas - saiba como funcionam as moedas no mercado de divisas.
O mercado forex funciona através de pares de moedas. Para simplificar, pense nisso como uma comparação entre o valor de uma moeda e o valor de outra, na forma de uma cotação.
Entendendo os tipos de gráficos básicos
O que é um gráfico? Quais são os tipos de gráficos mais comuns? Como eles diferem? Você sabia que todos os mercados financeiros geram dados? Clique para entender as diferentes maneiras como os dados são exibidos.
O que é um pip no mercado de divisas?
Um pip, ou porcentagem em pontos, representa a unidade de troca de um par de moedas, que é igual a um centésimo de um por cento. Ela representa o menor movimento de preços que qualquer taxa de câmbio pode ter.
