Cours vidéo les plus populaires
Paires de devises : Découvrez comment les devises fonctionnent sur le forex
Le marché des changes fonctionne au moyen de paires de devises. Pour le rendre plus simple, considérez-le comme comparant la valeur d'une devise à celle d'une autre, sous la forme d'une cotation.
Comprendre les types de graphiques de base
Qu'est-ce qu'un graphique ? Quels sont les types de graphiques les plus courants ? Comment sont-ils différents? Saviez-vous que tous les marchés financiers génèrent des données ? Cliquez pour comprendre les différentes manières dont les données sont affichées.
Qu'est-ce qu'un pip sur le marché des changes ?
Un pip, ou un pourcentage en points, représente l'unité de variation d'une paire de devises, égale à 1/100e de 1 %. Il représente le plus petit mouvement de prix qu'un taux de change puisse produire.
